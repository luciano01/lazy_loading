# Lazy Loading
Simple lazy loading with Mobx and (other branch) BLoC

## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/lazy_loading.git
$ cd lazy_loading
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd lazy_loading
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
